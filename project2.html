    • Overview

    This project is about the hangam game where the user picks the theme which can give them a better
    chance to guess the word The game will have an option of 5 themes which will be randomly selected.

    End Users

    This game is targeted towards 5-12 year olds. The choice of themes is highly likely to attract
    people at that age group. Children around those ages like to play games that are simple like hangman.

    • Specification

    (Introduction)

    An indroduction system will help the user identify what are they playing
    and what they can expect through out the game.

    (Tutorial)

    This will guide the user by telling what to do and show them how to
    play the game. This will help the user continue if they get stuck.

    (Win/Loss checker)

    Will tell the user if they have correctly or incorrectly guessed the letter. Helps the user
    to pick their new letter and figure out what they did wrong

    (Conclusion)

    Help the user identify that the game is over and will ask them again if the want to
    replay the game. If the answer No the game will be over
